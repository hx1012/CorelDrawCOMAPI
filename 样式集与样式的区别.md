# 样式（Style）与样式集（StyleSet）的区别

> 内容来源：《CorelDRAW X7 脚本参考手册》第 5 册（本仓库 PDF）

---

## 一、概念对比

| 对比项 | 样式（Style） | 样式集（StyleSet） |
|--------|-------------|------------------|
| **覆盖范围** | **单一类别**（fill / outline / character / paragraph / frame 之一） | **所有类别**（fill + outline + character + paragraph + frame 全部打包在一起） |
| **创建方法** | `StyleSheet.CreateStyle(Category, ...)` | `StyleSheet.CreateStyleSet(...)` |
| **必填参数** | `Category`（类别名称）为必填 | 无必填参数（没有 Category 的概念） |
| **返回类型** | `Style` 对象 | `Style` 对象（同一个类，但 `IsStyleSet = True`） |
| **区分方式** | `style.IsStyleSet = False` | `style.IsStyleSet = True` |
| **集合入口** | `StyleSheet.Styles` / `StyleSheet.AllStyles` | `StyleSheet.StyleSets` / `StyleSheet.AllStyleSets` |
| **UI 对应** | 在"对象样式"泊坞窗中的单一属性样式（如颜色样式、轮廓样式） | 在"对象样式"泊坞窗中的"图形和文本"样式集条目 |

---

## 二、创建方法签名对比

```vba
' ── 创建样式（必须指定 Category）─────────────────────────────────────
Function CreateStyle(
    Category As String,            ' 必填：类别名（"fill" / "outline" / "character" / "paragraph" / "frame"）
    [BasedOn As String],           ' 可选：继承自哪个样式
    [Name As String],              ' 可选：样式名称
    [ReplaceExisting As Boolean]   ' 可选：同名时是否覆盖
) As Style

' ── 创建样式集（无 Category 参数）────────────────────────────────────
Function CreateStyleSet(
    [BasedOn As String],           ' 可选：继承自哪个样式集
    [Name As String],              ' 可选：样式集名称
    [ReplaceExisting As Boolean]   ' 可选：同名时是否覆盖
) As Style
```

---

## 三、Category（类别）说明

`CreateStyle` 的 `Category` 参数决定这个样式控制的属性类别：

| Category 字符串 | 对应属性 | `Style` 上的入口属性 |
|----------------|---------|------------------|
| `"fill"` | 填充（颜色/渐变/图案） | `style.Fill`（`StyleFill` 对象） |
| `"outline"` | 轮廓线（线宽/颜色） | `style.Outline`（`StyleOutline` 对象） |
| `"character"` | 字符（字体/字号/粗斜体） | `style.Character`（`StyleCharacter` 对象） |
| `"paragraph"` | 段落（对齐/行间距） | `style.Paragraph`（`StyleParagraph` 对象） |
| `"frame"` | 文本框（列/边距） | `style.Frame`（`StyleFrame` 对象） |

> **样式集不含 Category**，其 `.Fill`、`.Outline`、`.Character` 等属性均可同时访问和设置。

## 四、集合的区别

```vba
Dim sheet As StyleSheet
Set sheet = ActiveDocument.StyleSheet

' 顶层"样式"列表（不含样式集）
Dim styles As Styles
Set styles = sheet.Styles         ' 顶层样式
Set styles = sheet.AllStyles      ' 全部样式（含嵌套/派生）

' 顶层"样式集"列表（不含普通样式）
Dim sets As Styles
Set sets = sheet.StyleSets        ' 顶层样式集
Set sets = sheet.AllStyleSets     ' 全部样式集（含嵌套/派生）

' 区分：遍历所有样式时通过 IsStyleSet 判断
Dim s As Style
For Each s In sheet.AllStyles
    If s.IsStyleSet Then
        Debug.Print "样式集：" & s.Name
    Else
        Debug.Print "样式：" & s.Name & "（类别：" & s.CategoryName & "）"
    End If
Next
```

## 五、代码示例

### 5.1 创建单一类别的"样式"并设置填充颜色

```vba
Dim sheet As StyleSheet
Set sheet = ActiveDocument.StyleSheet

' 创建 fill 类别的样式（必须指定类别）
Dim st As Style
Set st = sheet.CreateStyle("fill", "", "红色填充", True)

' ⚠️ 关键：必须先激活填充类型，否则颜色属性无效
st.Fill.Type = cdrUniformFillStyle   ' 1 = 纯色填充（cdrFillStyleType 枚举）

' 然后通过 Color 对象的方法（而不是直接赋值）设置颜色
st.Fill.PrimaryColor.RGBAssign 255, 0, 0   ' 红色

' 应用到形状（只影响填充，轮廓保持不变）
ActiveShape.ApplyStyle "红色填充"
```

### 5.2 创建"样式集"并同时设置填充 + 轮廓

```vba
Dim sheet As StyleSheet
Set sheet = ActiveDocument.StyleSheet

' 创建样式集（不需要 Category 参数）
Dim ss As Style
Set ss = sheet.CreateStyleSet("", "我的图形样式", True)

' 同时设置多个类别的属性
ss.Fill.Type = cdrUniformFillStyle
ss.Fill.PrimaryColor.RGBAssign 0, 128, 255      ' 蓝色填充

ss.Outline.Width = 0.5                           ' 0.5 mm 轮廓

' 应用到形状（fill + outline 同时生效）
ActiveShape.ApplyStyle "我的图形样式"
```

### 5.3 从现有形状捕获——推荐方式

与其手动初始化 `Fill.Type`，更稳妥的方式是从已设置好格式的形状直接捕获：

```vba
' 从当前形状捕获单一类别的样式
Dim newStyles As Styles
Set newStyles = sheet.CreateStyleFromShape(ActiveShape, "fill", "捕获填充样式", True)

' 从当前形状捕获样式集（fill + outline + character 全部一起）
Set newStyles = sheet.CreateStyleSetFromShape(ActiveShape, "捕获样式集", True)
```

## 六、导入/导出时的三类概念

`StyleSheet.Export` 的参数清楚地展示了三个独立的范畴：

```vba
' Styles      = 普通样式（单一类别）
' StyleSets   = 样式集（多类别打包）
' ObjectDefaults = 对象默认样式集（"Default Graphic" 等内置默认）
sheet.Export "my_styles.cdr", Styles:=True, StyleSets:=True, ObjectDefaults:=False
```

| 参数 | 说明 |
|------|------|
| `Styles` | 普通样式，`IsStyleSet = False` |
| `StyleSets` | 样式集，`IsStyleSet = True`，`IsObjectDefaults = False` |
| `ObjectDefaults` | 特殊样式集，`IsObjectDefaults = True`（如"Default Graphic"、"Default Artistic Text"） |

---

## 七、`IsObjectDefaults` —— 内置默认样式集

CorelDRAW 内部维护了几个特殊样式集作为各类对象的"出厂默认值"，可通过以下方式访问：

```vba
Dim defaults As Styles
Set defaults = ActiveDocument.StyleSheet.ObjectDefaults

Dim s As Style
For Each s In defaults
    Debug.Print s.Name   ' 如 "Default Graphic", "Default Artistic Text", "Default Paragraph Text"
Next
```

> `IsObjectDefaults = True` 的样式集不会出现在普通的 `StyleSets` 集合里，需要通过 `ObjectDefaults` 属性单独访问。

---

## 八、总结：三行核心结论

1. **样式（Style）** = 单类别（fill / outline / character / paragraph / frame 选其一），用 `CreateStyle(Category, ...)` 创建。
2. **样式集（StyleSet）** = 多类别打包，用 `CreateStyleSet(...)` 创建，同一个 `Style` 类，`IsStyleSet = True`。
3. **两者均用 `Shape.ApplyStyle(Name)` 应用**，区别在于：样式只更改对应的一个类别属性，样式集同时更改所有类别属性。
