# 如何通过脚本设置字体样式

> 内容来源：《CorelDRAW X7 脚本参考手册》第 3、5 册（本仓库 PDF）

---

## 一、文本对象的访问路径

操作字体样式必须先获取 **`TextRange`** 对象，访问链如下：

```vba
' ① 创建文本形状
Dim s As Shape
Set s = ActiveLayer.CreateParagraphText(2, 2, 8, 8, "这是示例文字。")
' 或
Set s = ActiveLayer.CreateArtisticText(0, 0, "这是示例文字。")

' ② 取 Text 对象
Dim t As Text
Set t = s.Text

' ③ Text.Story 是整段文字的 TextRange（最常用入口）
Dim tr As TextRange
Set tr = t.Story

' ④ 也可取子范围
Set tr = t.Story.Range(1, 4)          ' 第 1~4 个字符
Set tr = t.Story.Words.First          ' 第一个词
Set tr = t.Story.Words.Last           ' 最后一个词
Set tr = t.Story.Characters.All       ' 全部字符（等价于 Story 本身）
```

## 二、字体名称与字号（Font / Size）

**手册参考**：第 5 册第 737 页（`TextRange.Font`）、第 781 页（`TextRange.Size`）

```vba
' 设置字体名称（Font As String）
t.Story.Font = "Arial"

' 设置字号（Size As Single，单位：磅 points）
t.Story.Size = 24

' 在创建时直接通过可选参数指定字体和字号
Set s = ActiveLayer.CreateArtisticText(0, 0, "page: " & p.Index, _
        cdrEnglishUS, , "Arial", 6, cdrTrue, cdrTrue, , cdrLeftAlignment)
'                              ↑字体  ↑字号  ↑粗体    ↑斜体
```

## 三、粗体 / 斜体 / 删除线（Bold / Italic / Strikethru）

**手册参考**：第 5 册第 706 页（Bold）、第 755 页（Italic）、第 784 页（Strikethru）

```vba
' 粗体（Bold As Boolean）
t.Story.Bold = True

' 斜体（Italic As Boolean）
t.Story.Italic = True

' 删除线（Strikethru As cdrFontLine）
t.Story.Strikethru = cdrDoubleThinFontLine

' 也可操作子范围
t.Story.Words.Last.Bold = True
t.Story.Words.First.Italic = True
t.Story.Range(4, 7).Bold = True   ' 第 4~7 个字符加粗
```

## 四、字体样式枚举（Style — 综合粗斜体枚举）

**手册参考**：第 5 册第 785 页（`TextRange.Style`）

`Style` 属性使用 `cdrFontStyle` 枚举，可以一次指定粗体/斜体/粗细等级：

```vba
' Style As cdrFontStyle
t.Story.Style = cdrBoldFontStyle          ' 粗体
t.Story.Style = cdrItalicFontStyle        ' 斜体
t.Story.Style = cdrBoldItalicFontStyle    ' 粗斜体
t.Story.Style = cdrNormalFontStyle        ' 恢复正常
```

### `cdrFontStyle` 枚举常量

| 常量 | 值 | 说明 |
|------|----|------|
| `cdrNormalFontStyle` | 0 | 常规 |
| `cdrBoldFontStyle` | 1 | 粗体 |
| `cdrItalicFontStyle` | 2 | 斜体 |
| `cdrBoldItalicFontStyle` | 3 | 粗斜体 |
| `cdrThinFontStyle` | 4 | 细体（Thin） |
| `cdrThinItalicFontStyle` | 5 | 细斜体 |
| `cdrExtraLightFontStyle` | 6 | 超细（ExtraLight） |
| `cdrExtraLightItalicFontStyle` | 7 | 超细斜体 |
| `cdrMediumFontStyle` | 8 | 中等（Medium） |
| `cdrMediumItalicFontStyle` | 9 | 中等斜体 |
| `cdrSemiBoldFontStyle` | 10 | 半粗（SemiBold） |
| `cdrSemiBoldItalicFontStyle` | 11 | 半粗斜体 |
| `cdrExtraBoldFontStyle` | 12 | 超粗（ExtraBold） |
| `cdrExtraBoldItalicFontStyle` | 13 | 超粗斜体 |
| `cdrHeavyFontStyle` | 14 | 黑体（Heavy） |
| `cdrHeavyItalicFontStyle` | 15 | 黑体斜体 |
| `cdrLightFontStyle` | 17 | 细体（Light） |
| `cdrLightItalicFontStyle` | 18 | 细斜体 |
| `cdrMixedFontStyle` | 16 | 混合（多种样式混用时） |

## 五、下划线（Underline）

**手册参考**：第 5 册第 790 页（`TextRange.Underline`）

```vba
' Underline As cdrFontLine
t.Story.Underline = cdrSingleThinFontLine      ' 细实线
t.Story.Underline = cdrSingleThickFontLine     ' 粗实线
t.Story.Underline = cdrDoubleThinFontLine      ' 双细线
t.Story.Underline = cdrNoFontLine              ' 取消下划线

' 配合 SetRange 对局部文字加下划线
Dim tr As TextRange
Set tr = s.Text.Story
tr.SetRange 3, 10                              ' 选第 3~10 个字符
tr.Underline = cdrDoubleThinFontLine
```

### `cdrFontLine` 枚举常量（下划线 / 删除线通用）

| 常量 | 值 | 说明 |
|------|----|------|
| `cdrNoFontLine` | 0 | 无线 |
| `cdrSingleThinFontLine` | 1 | 细实线 |
| `cdrSingleThinWordFontLine` | 2 | 细实线（含空格） |
| `cdrSingleThickFontLine` | 3 | 粗实线 |
| `cdrSingleThickWordFontLine` | 4 | 粗实线（含空格） |
| `cdrDoubleThinFontLine` | 5 | 双细线 |
| `cdrDoubleThinWordFontLine` | 6 | 双细线（含空格） |
| `cdrMixedFontLine` | 7 | 混合 |

## 六、大小写（Case / ChangeCase）

**手册参考**：第 5 册第 713 页（`TextRange.Case`）、第 714 页（`TextRange.ChangeCase`）

```vba
' Case As cdrFontCase — 控制显示样式（不改变实际字符存储）
t.Story.Case = cdrAllCapsFontCase      ' 全大写显示
t.Story.Case = cdrSmallCapsFontCase    ' 小型大写字母
t.Story.Case = cdrNormalFontCase       ' 恢复正常

' ChangeCase — 真正改变字符内容
t.Story.ChangeCase cdrTextUpperCase    ' 转为全大写
t.Story.ChangeCase cdrTextLowerCase    ' 转为全小写
```

### `cdrFontCase` 枚举常量

| 常量 | 值 | 说明 |
|------|----|------|
| `cdrNormalFontCase` | 0 | 正常 |
| `cdrSmallCapsFontCase` | 1 | 小型大写 |
| `cdrAllCapsFontCase` | 2 | 全大写 |
| `cdrMixedFontCase` | 3 | 混合 |

## 八、字体颜色（Fill）

**手册参考**：第 5 册第 734 页（`TextRange.Fill`）

```vba
' 纯色（RGB）
t.Story.Fill.ApplyUniformFill CreateRGBColor(255, 0, 0)       ' 红色

' 对单个词应用渐变色
t.Story.Words.Last.Fill.ApplyFountainFill _
    CreateRGBColor(255, 0, 0), CreateRGBColor(0, 0, 255), _
    cdrConicalFountainFill
```

## 十、字符间距 / 词间距 / 行间距

**手册参考**：第 5 册第 719 页（CharSpacing）、第 794 页（WordSpacing）、第 759-760 页（LineSpacing）

```vba
' 字符间距（CharSpacing As Single）
t.Story.CharSpacing = 1.3

' 词间距（WordSpacing As Single）
t.Story.WordSpacing = 2.0

' 行间距（LineSpacing As Single，只读；用 SetLineSpacing 设置）
MsgBox t.Story.LineSpacing

' 精确设置行间距（含类型）
t.Story.SetLineSpacing cdrPercentOfCharacterHeightLineSpacing, 1.2, 0.2, 0.4
```

### `cdrLineSpacingType` 枚举常量

| 常量 | 值 | 说明 |
|------|----|------|
| `cdrPercentOfCharacterHeightLineSpacing` | 0 | 按字符高度的百分比 |
| `cdrPointLineSpacing` | 1 | 固定磅值 |
| `cdrPercentOfPointSizeLineSpacing` | 2 | 按字号的百分比 |
| `cdrMixedLineSpacing` | 3 | 混合 |

## 十一、字偶间距（RangeKerning）与字符旋转

```vba
' 字符范围级别的字偶间距（RangeKerning As Long）
t.Story.Words.First.RangeKerning = -20   ' 负值收紧，正值加宽

' 字符旋转角度（CharAngle As Single，度）
t.Story.CharAngle = 15.0

' 水平 / 垂直位移（HorizShift / VertShift As Long）
t.Story.HorizShift = 3
t.Story.VertShift = 3
```

## 十三、综合示例

```vba
Sub FontStyleDemo()
    If Application.Documents.Count = 0 Then CreateDocument

    Dim s As Shape
    Dim t As Text

    ' 创建段落文本
    Set s = ActiveLayer.CreateParagraphText(2, 2, 10, 8, _
            "Hello World! This is a CorelDRAW font demo.")
    Set t = s.Text

    ' 全文：Arial 字体，14 号，左对齐
    t.Story.Font      = "Arial"
    t.Story.Size      = 14
    t.Story.Alignment = cdrLeftAlignment

    ' 第一个词：粗体 + 蓝色
    t.Story.Words.First.Bold = True
    t.Story.Words.First.Fill.ApplyUniformFill CreateRGBColor(0, 0, 200)

    ' 第二个词：斜体
    t.Story.Words(2, 2).Italic = True

    ' 第 3~7 个字符：双细下划线
    Dim tr As TextRange
    Set tr = t.Story
    tr.SetRange 3, 7
    tr.Underline = cdrDoubleThinFontLine

    ' 最后一个词：上标
    t.Story.Words.Last.Position = cdrSuperscriptFontPosition

    ' 字符间距
    t.Story.CharSpacing = 0.5

    ' 行间距：按字符高度的 120%
    t.Story.SetLineSpacing cdrPercentOfCharacterHeightLineSpacing, 1.2, 0, 0

    s.Selected = True
    MsgBox "字体样式设置完成！", vbInformation
End Sub
```
