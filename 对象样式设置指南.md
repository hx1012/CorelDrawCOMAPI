## 一、轮廓（Outline）

**手册参考**：第 4 册 `IVGOutline` 类；第 3 册 `Shape.Outline` 属性

`Shape.Outline` 返回一个 **`IVGOutline`** 对象，用于控制形状的描边（轮廓线）。

### 1.1 基本用法

```vba
Dim s As Shape
Set s = ActiveLayer.CreateRectangle(10, 10, 80, 60)   ' 创建矩形

' 获取 Outline 对象（也可直接赋值属性）
Dim ol As IVGOutline
Set ol = s.Outline

' 线宽（文档单位，默认 mm）
ol.Width = 1.0

' 线色（RGB）
ol.Color.RGBAssign 200, 50, 50

' 取消轮廓
s.Outline.Width = 0
```

### 1.2 线型（Style / DashDotPattern）

```vba
' 使用预设虚线样式（cdrOutlineStyle 枚举）
ol.Style = cdrDashDotOutlineStyle      ' 点划线
ol.Style = cdrDashedOutlineStyle       ' 虚线
ol.Style = cdrDottedOutlineStyle       ' 点线
ol.Style = cdrSolidOutlineStyle        ' 实线（默认）

' 自定义虚线——通过 DashDotPattern 属性（返回 IVGDashDotPattern）
' 需先调用 CustomDashPattern 方法
```

### 1.3 线端 / 线角（LineCaps / LineJoin）

```vba
' 线端样式（cdrLineCapType）
ol.StartArrow = Nothing                ' 无箭头
ol.EndArrow   = Nothing

' 端头形状
ol.LineCaps = cdrRoundLineCap          ' 圆头
ol.LineCaps = cdrSquareLineCap         ' 方头（默认）
ol.LineCaps = cdrFlatLineCap           ' 平头

' 转角样式（cdrLineJoinType）
ol.LineJoin = cdrRoundLineJoin         ' 圆角
ol.LineJoin = cdrMiterLineJoin         ' 尖角
ol.LineJoin = cdrBevelLineJoin         ' 斜切
```

### 1.4 箭头

```vba
' 箭头来自 Application.ArrowHeads 集合，通过索引取得
ol.StartArrow = Application.ArrowHeads(3)   ' 起点箭头
ol.EndArrow   = Application.ArrowHeads(7)   ' 终点箭头
ol.EndArrow   = Nothing                      ' 清除箭头
```

### 1.5 "后置于对象之后"

```vba
' ScaleWithShape：轮廓随形状缩放
ol.ScaleWithShape = True
```

### 1.6 轮廓颜色模式

```vba
' RGB
ol.Color.RGBAssign 255, 128, 0

' CMYK
ol.Color.CMYKAssign 0, 50, 100, 0

' 灰度
ol.Color.GrayAssign 128

' 套版色（Registration Color，用于印刷裁剪线）
ol.Color.RegistrationAssign
```

### 1.7 枚举速查

| 枚举 | 常用常量 |
|------|----------|
| `cdrOutlineStyle` | `cdrSolidOutlineStyle`, `cdrDashedOutlineStyle`, `cdrDottedOutlineStyle`, `cdrDashDotOutlineStyle` |
| `cdrLineCapType` | `cdrFlatLineCap`, `cdrRoundLineCap`, `cdrSquareLineCap` |
| `cdrLineJoinType` | `cdrMiterLineJoin`, `cdrRoundLineJoin`, `cdrBevelLineJoin` |

## 二、填充（Fill）

**手册参考**：第 4 册 `Fill` 类；第 3 册 `Shape.Fill` 属性

`Shape.Fill` 返回一个 **`Fill`** 对象，支持均匀色、渐变、图样、纹理、PostScript 五种类型。

### 2.1 均匀填充（Uniform Fill）

```vba
Dim s As Shape
Set s = ActiveLayer.CreateEllipse(10, 10, 80, 60)

' 方式一：直接 ApplyUniformFill
s.Fill.ApplyUniformFill CreateRGBColor(31, 73, 125)

' 方式二：使用 Color 对象
Dim c As New Color
c.RGBAssign 220, 30, 30
s.Fill.ApplyUniformFill c

' CMYK 颜色
Dim c2 As New Color
c2.CMYKAssign 0, 100, 100, 0   ' 纯红
s.Fill.ApplyUniformFill c2

' 取消填充（无填充）
s.Fill.ApplyNoFill
```

### 2.2 渐变填充（Fountain Fill）

```vba
' 线性渐变（两色）
s.Fill.ApplyFountainFill _
    CreateRGBColor(255, 255, 0), _   ' 起始色（黄）
    CreateRGBColor(255, 0, 0), _     ' 终止色（红）
    cdrLinearFountainFill            ' 渐变类型

' 辐射渐变
s.Fill.ApplyFountainFill _
    CreateRGBColor(255, 255, 255), _
    CreateRGBColor(0, 0, 0), _
    cdrRadialFountainFill

' 锥形渐变
s.Fill.ApplyFountainFill _
    CreateRGBColor(255, 0, 0), _
    CreateRGBColor(0, 0, 255), _
    cdrConicalFountainFill

' 方形渐变
s.Fill.ApplyFountainFill _
    CreateRGBColor(0, 200, 0), _
    CreateRGBColor(0, 50, 0), _
    cdrSquareFountainFill

' 高级参数（角度、中心位移、中点等）
Dim ff As FountainFill
Set ff = s.Fill.Fountain
ff.Angle    = 45       ' 渐变角度（度）
ff.CenterX  = 20       ' 中心 X 偏移（%）
ff.CenterY  = 20       ' 中心 Y 偏移（%）
ff.MidPoint = 60       ' 中点位置（%，1-99）
ff.Steps    = 256      ' 渐变步长

' 多色渐变（FountainFillStop 集合）
ff.Stops.Add CreateRGBColor(255, 255, 0), 0    ' 位置 0%  = 黄
ff.Stops.Add CreateRGBColor(255, 128, 0), 50   ' 位置 50% = 橙
ff.Stops.Add CreateRGBColor(255, 0, 0), 100    ' 位置 100% = 红
```

### `cdrFountainFillType` 枚举

| 常量 | 说明 |
|------|------|
| `cdrLinearFountainFill` | 线性 |
| `cdrRadialFountainFill` | 辐射 |
| `cdrConicalFountainFill` | 锥形 |
| `cdrSquareFountainFill` | 方形 |

### 2.3 双色图样填充（Two-Color Pattern Fill）

```vba
' ApplyTwoColorPatternFill(PatternIndex, FrontColor, BackColor)
s.Fill.ApplyTwoColorPatternFill 2, _
    CreateRGBColor(0, 0, 0), _
    CreateRGBColor(255, 255, 255)

' 图样索引 1~n 对应 Application.PatternFills 集合
```

### 2.4 全色图样填充（Full-Color Pattern Fill）

```vba
' 需要图样文件路径（.pat / .cdr 等）
s.Fill.ApplyFullColorPatternFill "C:\Patterns\mypattern.pat"

' 或通过内置图样集合
s.Fill.ApplyFullColorPatternFill Application.PatternFills(5).FullPath
```

### 2.5 纹理填充（Texture Fill）

```vba
' ApplyTextureFill(TextureName, TextureLibrary)
s.Fill.ApplyTextureFill "Rainstorm", "Samples7"

' 列出可用库和纹理
Dim lib As TextureLibrary
For Each lib In Application.TextureLibraries
    Debug.Print lib.Name
Next lib
```

### 2.6 PostScript 填充

```vba
' ApplyPostScriptFill(PatternName)
s.Fill.ApplyPostScriptFill "Camouflage"
```

### 2.7 填充类型判断

```vba
Select Case s.Fill.Type
    Case cdrNoFill              : Debug.Print "无填充"
    Case cdrUniformFill         : Debug.Print "均匀"
    Case cdrFountainFill        : Debug.Print "渐变"
    Case cdrPatternFill         : Debug.Print "图样"
    Case cdrTextureFill         : Debug.Print "纹理"
    Case cdrPostScriptFill      : Debug.Print "PostScript"
End Select
```

## 三、透明度（Transparency）

**手册参考**：第 4 册 `Transparency` 类；`Shape.Transparency` 属性

透明度与填充的 API 结构类似，但通过 **`Shape.Transparency`** 访问。

### 3.1 均匀透明度

```vba
Dim s As Shape
Set s = ActiveLayer.CreateRectangle(20, 20, 100, 70)

' 设置均匀透明度（0 = 不透明，100 = 完全透明）
s.Transparency.ApplyUniformTransparency 50   ' 半透明

' 取消透明度
s.Transparency.ApplyNoTransparency
```

### 3.2 渐变透明度（Fountain Transparency）

```vba
' 类似渐变填充，但用 ApplyFountainTransparency
s.Transparency.ApplyFountainTransparency _
    50, 0, _                       ' 起始透明度、终止透明度
    cdrLinearFountainFill, _       ' 渐变类型
    90, 0, 0, 256                  ' 角度、CenterX、CenterY、Steps
```

### 3.3 图样透明度（Pattern Transparency）

```vba
' 双色图样
s.Transparency.ApplyTwoColorPatternTransparency 3, 80, 20
'                                               ↑图样  ↑前景透明度 ↑背景透明度
```

### 3.4 纹理透明度

```vba
s.Transparency.ApplyTextureTransparency "Rainstorm", "Samples7"
```

### 3.5 合并模式（Merge Mode）

```vba
' MergeMode 控制与下层对象的混合方式
s.Transparency.MergeMode = cdrNormalMergeMode       ' 正常
s.Transparency.MergeMode = cdrMultiplyMergeMode     ' 正片叠底
s.Transparency.MergeMode = cdrScreenMergeMode       ' 屏幕
s.Transparency.MergeMode = cdrOverlayMergeMode      ' 叠加
s.Transparency.MergeMode = cdrDifferenceMergeMode   ' 差值
```

### `cdrMergeMode` 常用枚举常量

| 常量 | 说明 |
|------|------|
| `cdrNormalMergeMode` | 正常 |
| `cdrMultiplyMergeMode` | 正片叠底 |
| `cdrScreenMergeMode` | 屏幕 |
| `cdrOverlayMergeMode` | 叠加 |
| `cdrHardLightMergeMode` | 强光 |
| `cdrSoftLightMergeMode` | 柔光 |
| `cdrDifferenceMergeMode` | 差值 |
| `cdrColorMergeMode` | 颜色 |
| `cdrLuminanceMergeMode` | 亮度 |

## 四、字符样式（Character）

> 详细内容见 [`字体样式设置指南.md`](字体样式设置指南.md)。本节补充与对象样式系统集成相关的 API。

### 4.1 对象级别字体属性快速设置

```vba
' Shape 创建时直接传入字体参数
Dim s As Shape
Set s = ActiveLayer.CreateArtisticText(10, 10, "CorelDRAW", _
        cdrChineseSimplified, , "黑体", 36, _
        cdrTrue,  _   ' 粗体
        cdrFalse, _   ' 斜体
        , cdrCenterAlignment)

' 或事后通过 Text.Story 统一设置
With s.Text.Story
    .Font      = "微软雅黑"
    .Size      = 24
    .Bold      = True
    .Fill.ApplyUniformFill CreateRGBColor(0, 80, 180)
End With
```

### 4.2 通过对象样式库（ObjectStyles）应用命名样式

```vba
' 将当前文档中名为 "Heading1" 的字符样式应用到文本
Dim cs As ObjectStyle
Set cs = ActiveDocument.ObjectStyles("Heading1")
cs.Apply s
```

### 4.3 字符位置与变换

```vba
Dim tr As TextRange
Set tr = s.Text.Story

' 字符旋转
tr.CharAngle = 30.0

' 字符水平 / 垂直偏移（设计单位）
tr.HorizShift = 2
tr.VertShift   = 1

' 字偶间距（负值收紧）
tr.RangeKerning = -15
```

## 五、段落样式（Paragraph）

**手册参考**：第 5 册 `TextRange` 段落相关属性

段落属性均通过 **`TextRange`** 对象（`Shape.Text.Story` 或段落子范围）访问。

### 5.1 缩进（Indent）

```vba
Dim tr As TextRange
Set tr = s.Text.Story

' 首行缩进（FirstLineIndent，单位：文档单位）
tr.FirstLineIndent = 5.0     ' 5 mm

' 左缩进
tr.LeftIndent = 10.0         ' 10 mm

' 右缩进
tr.RightIndent = 5.0         ' 5 mm
```

### 5.2 段前 / 段后间距（SpaceBefore / SpaceAfter）

```vba
' 段前间距（SpaceBefore，单位：磅或文档单位）
tr.SpaceBefore = 6.0

' 段后间距（SpaceAfter）
tr.SpaceAfter = 4.0
```

### 5.3 行间距（LineSpacing）

```vba
' 精确设置（手册 SetLineSpacing 方法）
' 参数：类型, LineSpace, SpaceBefore, SpaceAfter
tr.SetLineSpacing cdrPercentOfCharacterHeightLineSpacing, 1.5, 0.3, 0.3
' 固定磅值
tr.SetLineSpacing cdrPointLineSpacing, 18.0, 0, 0
```

### 5.4 对齐（Alignment）

```vba
tr.Alignment = cdrLeftAlignment            ' 左对齐
tr.Alignment = cdrCenterAlignment          ' 居中
tr.Alignment = cdrRightAlignment           ' 右对齐
tr.Alignment = cdrFullJustifyAlignment     ' 两端对齐
tr.Alignment = cdrForceJustifyAlignment    ' 强制两端
tr.Alignment = cdrNoneAlignment            ' 无对齐（美术字默认）
```

### 5.5 制表位（Tabs）

```vba
Dim tp As TextRange
Set tp = s.Text.Story

' 添加自定义制表位（位置 mm，对齐方式）
tp.Tabs.Add 20.0, cdrLeftTabAlignment
tp.Tabs.Add 60.0, cdrCenterTabAlignment
tp.Tabs.Add 100.0, cdrRightTabAlignment
tp.Tabs.Add 140.0, cdrDecimalTabAlignment

' 清除所有制表位
tp.Tabs.Clear
```

### 5.6 分栏

```vba
' 段落文本框分栏通过 TextFrame 对象设置（见第六节）
```

### 5.7 字符间距 / 词间距

```vba
tr.CharSpacing = 0.8    ' 字符间距（比例）
tr.WordSpacing = 1.5    ' 词间距（比例）
```

### 5.8 段落枚举速查

| 属性 / 方法 | 类型 | 说明 |
|-------------|------|------|
| `Alignment` | `cdrAlignment` | 对齐方式 |
| `FirstLineIndent` | `Single` | 首行缩进（mm） |
| `LeftIndent` | `Single` | 左缩进 |
| `RightIndent` | `Single` | 右缩进 |
| `SpaceBefore` | `Single` | 段前间距 |
| `SpaceAfter` | `Single` | 段后间距 |
| `SetLineSpacing` | 方法 | 行间距（类型+值） |
| `LineSpacing` | `Single`（只读） | 当前行间距值 |
| `CharSpacing` | `Single` | 字符间距 |
| `WordSpacing` | `Single` | 词间距 |
| `Tabs.Add` | 方法 | 添加制表位 |
| `Tabs.Clear` | 方法 | 清除制表位 |

## 六、图文框（Text Frame）

**手册参考**：第 5 册 `TextFrame` 类；`Shape.TextFrame` 属性

图文框（Frame）是段落文本形状（`CreateParagraphText` 创建）特有的容器对象。

### 6.1 访问 TextFrame

```vba
Dim s As Shape
Set s = ActiveLayer.CreateParagraphText(10, 10, 120, 80, "")

Dim tf As TextFrame
Set tf = s.TextFrame     ' 访问 TextFrame 对象
```

### 6.2 垂直对齐（VertAlign）

```vba
' VertAlign 控制文字在框内的垂直位置
tf.VertAlign = cdrTopTextAlignment     ' 顶部对齐（默认）
tf.VertAlign = cdrMiddleTextAlignment  ' 垂直居中
tf.VertAlign = cdrBottomTextAlignment  ' 底部对齐
tf.VertAlign = cdrFullJustifyTextAlignment  ' 垂直两端
```

### 6.3 分栏（Columns）

```vba
' 设置列数
tf.ColumnCount = 3

' 设置每列宽度（如等宽）
Dim colWidth As Double
colWidth = (s.SizeWidth - (tf.ColumnCount - 1) * tf.ColumnGutter) / tf.ColumnCount
Dim i As Integer
For i = 1 To tf.ColumnCount
    tf.ColumnWidth(i) = colWidth
Next i

' 栏间距
tf.ColumnGutter = 5.0   ' 5 mm 间距
```

### 6.4 文本溢出（流入下一框）

```vba
' 段落文本可链接到下一个文本框（Linked Frame）
Dim s1 As Shape, s2 As Shape
Set s1 = ActiveLayer.CreateParagraphText(10, 10, 80, 60, "长文本...")
Set s2 = ActiveLayer.CreateParagraphText(90, 10, 160, 60, "")

' 将 s1 的溢出文本流入 s2
s1.TextFrame.LinkTo s2.TextFrame
```

### 6.5 框架类型 / 文字方向

```vba
' TextFrame 类型
tf.FrameType = cdrRectangularTextFrame      ' 矩形框
tf.FrameType = cdrCustomTextFrame           ' 异形框

' 文字排列方向
tf.HorzOrVert = cdrHorizontalTextFrame      ' 横排
tf.HorzOrVert = cdrVerticalTextFrame        ' 竖排
```

### 6.6 内边距（Padding / Margin）

```vba
' 内边距（文本距框边的距离）
tf.TopMargin    = 2.0
tf.BottomMargin = 2.0
tf.LeftMargin   = 3.0
tf.RightMargin  = 3.0
```

### 6.7 自动调整大小

```vba
' 使框随文本内容自动调整高度
tf.AutoExpand = True
```

## 七、位图效果（Bitmap Effects）

**手册参考**：第 4 册 `Shape.Effects` / `Bitmap` 类；第 7 册效果章节

### 7.1 将矢量转换为位图

```vba
Dim s As Shape
Set s = ActiveLayer.CreateEllipse(10, 10, 80, 60)
s.Fill.ApplyUniformFill CreateRGBColor(200, 100, 50)

' ConvertToBitmap(Resolution, AntiAlias, TransparentBackground, ColorMode, ApplyICCProfile)
Dim bmp As Shape
Set bmp = s.ConvertToBitmap(300, True, True, cdrRGBColorMode, False)
```

### 7.2 导入位图

```vba
Dim bmp As Shape
Set bmp = ActiveLayer.Import("C:\Images\photo.jpg")
' 或使用 Document.Import 然后获取新导入的形状
```

### 7.3 位图重采样 / 裁剪

```vba
' 重采样（改变分辨率/尺寸）
bmp.Bitmap.Resample 150, True, False
'                   ↑dpi   ↑保持纵横比  ↑仅改分辨率(不改像素数)

' 裁剪（按文档坐标）
bmp.Bitmap.Crop 20, 20, 100, 80
```

### 7.4 位图颜色模式转换

```vba
' ChangeColorMode(ColorMode)
bmp.Bitmap.ChangeColorMode cdrGrayscaleColorMode     ' 灰度
bmp.Bitmap.ChangeColorMode cdrRGBColorMode           ' RGB
bmp.Bitmap.ChangeColorMode cdrCMYKColorMode          ' CMYK
bmp.Bitmap.ChangeColorMode cdrBWColorMode            ' 黑白（1-bit）
```

### `cdrColorMode` 常量

| 常量 | 说明 |
|------|------|
| `cdrBWColorMode` | 黑白（1 位） |
| `cdrGrayscaleColorMode` | 灰度（8 位） |
| `cdrRGBColorMode` | RGB（24 位） |
| `cdrCMYKColorMode` | CMYK（32 位） |
| `cdrLabColorMode` | Lab 色彩空间 |

### 7.5 PowerClip（精确剪裁）

将内容放入容器形状。

```vba
Dim container As Shape   ' 容器（任意形状）
Dim content   As Shape   ' 要放入容器的对象

Set container = ActiveLayer.CreateEllipse(10, 10, 100, 80)
Set content   = ActiveLayer.CreateRectangle(20, 20, 90, 70)
content.Fill.ApplyUniformFill CreateRGBColor(255, 200, 0)

' 将 content 放入 container 的 PowerClip 内
container.PowerClip.PlaceContent content

' 访问 PowerClip 内部内容
Dim inner As Shape
Set inner = container.PowerClip.Contents.First

' 退出/进入编辑 PowerClip 内容
container.PowerClip.Edit
container.PowerClip.FinishEdit
```

### 7.6 透镜效果（Lens）

```vba
' 应用透镜（通过 Shape.Lens 属性）
Dim s As Shape
Set s = ActiveLayer.CreateEllipse(30, 30, 120, 90)

' 放大镜
s.Lens.ApplyLens cdrMagnifyLens
s.Lens.Magnify.Amount = 2.0      ' 放大倍率

' 鱼眼
s.Lens.ApplyLens cdrFishEyeLens
s.Lens.FishEye.Amount = 150.0

' 线框透镜
s.Lens.ApplyLens cdrWireframeLens
s.Lens.Wireframe.OutlineColor.RGBAssign 0, 0, 255

' 颜色滤镜
s.Lens.ApplyLens cdrColorLimitLens
s.Lens.ColorLimit.Color.RGBAssign 255, 0, 0
s.Lens.ColorLimit.Rate = 50

' 自定义颜色
s.Lens.ApplyLens cdrCustomColorLens

' 取消透镜
s.Lens.ApplyNoLens

' 冻结透镜内容（不再动态更新）
s.Lens.Frozen = True

' 仅显示透镜（不显示轮廓）
s.Lens.ViewPoint = True
```

### `cdrLensType` 常量

| 常量 | 说明 |
|------|------|
| `cdrMagnifyLens` | 放大镜 |
| `cdrFishEyeLens` | 鱼眼 |
| `cdrWireframeLens` | 线框 |
| `cdrColorLimitLens` | 颜色限制 |
| `cdrColorAddLens` | 颜色添加 |
| `cdrCustomColorLens` | 自定义颜色 |
| `cdrTransparencyLens` | 透明度 |
| `cdrInvertLens` | 反转 |
| `cdrTintLens` | 着色 |
| `cdrHeatMapLens` | 热力图 |

### 7.7 阴影（DropShadow）

```vba
' Shape.DropShadow 返回 DropShadow 对象
Dim s As Shape
Set s = ActiveLayer.CreateRectangle(20, 20, 100, 70)

Dim ds As DropShadow
Set ds = s.DropShadow

' 应用阴影
ds.Apply
ds.OffsetX   = 3.0                         ' 水平偏移（mm）
ds.OffsetY   = -3.0                        ' 垂直偏移（mm）
ds.Opacity   = 70                          ' 不透明度（%）
ds.Feather   = 5.0                         ' 羽化（mm）
```

## 九、样式表系统（StyleSheet）

> ⚠️ **重要纠错**：`Document` 对象 **没有** `ObjectStyles` 属性或方法，也不存在
> `ObjectStyle` 类型。管理命名样式的正确入口是 **`Document.StyleSheet`**，它返回
> 一个 `StyleSheet` 对象。

**手册参考**：第 2 册 `Document.StyleSheet`；第 5 册 `StyleSheet` / `Style` / `Styles` 类；
第 4 册 `Shape.ApplyStyle` / `Shape.Style`

---

### 9.1 对象模型层次

```
Document
  └── StyleSheet          (Document.StyleSheet，只读)
        ├── Styles         (顶层 Style 集合)
        ├── StyleSets      (顶层样式集 Style 集合，每个 Style.IsStyleSet = True)
        ├── AllStyles      (全部 Style，含派生)
        ├── AllStyleSets   (全部样式集，含派生)
        └── ObjectDefaults (默认对象样式集)

Style
  ├── Name / CategoryName / DisplayName
  ├── IsStyleSet / IsObjectDefaults
  ├── BasedOn / DerivedStyles
  ├── Outline  → StyleOutline  (轮廓属性)
  ├── Fill     → StyleFill     (填充属性)
  ├── Character → StyleCharacter (字符属性)
  ├── Paragraph → StyleParagraph (段落属性)
  └── Frame    → StyleFrame    (图文框属性)
```

---

### 9.2 样式（Style）与样式集（StyleSet）的区别

| | 样式（Style） | 样式集（StyleSet） |
|---|---|---|
| `IsStyleSet` | `False` | `True` |
| 覆盖范围 | 单一类别（如仅"填充"或仅"轮廓"） | 多类别组合（可同时包含填充、轮廓、字符等） |
| 创建方法 | `StyleSheet.CreateStyle(Category, ...)` | `StyleSheet.CreateStyleSet(...)` |
| 从形状创建 | `StyleSheet.CreateStyleFromShape(Shape, Category, ...)` | `StyleSheet.CreateStyleSetFromShape(Shape, ...)` |
| 应用到形状 | `Shape.ApplyStyle("样式名")` | `Shape.ApplyStyle("样式集名")` |

### 9.11 样式的继承（BasedOn）

```vba
    Dim ss As StyleSheet
    Set ss = ActiveDocument.StyleSheet

    ' ── 1. 建立填充样式 ──────────────────────────
    Dim fillSt As Style

    Set olSt = ss.CreateStyle("Outline", , "BrandOutline", True)

        s.ApplyStyle "BrandOutline"
    Next i

    fillSt.Fill.PrimaryColor.RGBAssign 200, 50, 50   ' 改为红色，三个矩形同步更新

    MsgBox "样式已应用并批量更新！"
End Sub
```

## 十、直接属性设置 vs. 样式表应用——区别与建议

### 10.1 核心区别

| 维度 | 直接属性设置 | 样式表（StyleSheet）应用 |
|------|-------------|------------------------|
| **方式** | `s.Fill.ApplyUniformFill(...)` / `s.Outline.Width = x` | `ss.CreateStyle(...)` → `s.ApplyStyle("名称")` |
| **作用范围** | 仅影响当前形状（本地覆盖） | 命名样式可跨形状共享 |
| **批量更新** | 需逐个形状修改 | 只需修改 `Style` 对象，所有使用该样式的形状自动更新 |
| **继承关系** | 无 | 可通过 `BasedOn` 建立父子继承链 |
| **持久化** | 随形状保存在文档中 | 随样式表保存，可导出 / 导入 / 设为默认 |
| **读回** | `s.Fill.Type`、`s.Outline.Width` 等 | `s.Style`（只读）返回当前 `Style` 对象 |
| **适用场景** | 一次性效果、原型探索 | 统一品牌规范、大量重复样式的文档 |

### 10.2 局部覆盖（Override）

对形状调用 `ApplyStyle` 后，直接在形状上修改属性（如 `s.Fill.ApplyUniformFill`）会产生**局部覆盖**，此时形状的外观独立于样式而存在。如果后续修改样式，被局部覆盖的属性将**不会**跟随样式更新。

```vba
' 应用样式后进行局部覆盖
s.ApplyStyle "BrandFill"          ' 样式：深蓝
s.Fill.ApplyUniformFill _
    CreateRGBColor(255, 100, 0)   ' 局部覆盖：橙色（脱离样式控制）

' 此时修改 BrandFill 样式颜色，不会影响 s 的填充（因为它已有本地覆盖）
```

### 10.3 选择建议

- **探索 / 单次效果** → 直接属性设置更简单快捷。  
- **多形状统一外观 / 品牌规范 / 需要批量修改** → 使用样式表，维护更高效。  
- **模板化工作流** → 用 `SaveStyleSheet` / `LoadStyleSheet` 在文档间共享样式。
