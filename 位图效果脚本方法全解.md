# CorelDRAW 脚本设置位图效果方法全解

> 内容来源：《CorelDRAW X7 脚本参考手册》第 1、3、4、5 册（本仓库 PDF）+ 官方社区文档

---

## 概览：四大类方法

| 类别 | 作用对象 | 典型入口 |
|------|---------|---------|
| **① 位图效果插件**（Bitmap Effects） | `cdrBitmapShape` 的位图内容 | `Shape.Bitmap.ApplyBitmapEffect` |
| **② 色调 / 颜色调整**（Color Adjustments） | 任意形状（含位图）| `Shape.ApplyEffect*` 系列方法 |
| **③ CorelDRAW 矢量特效**（Vector Effects） | 任意形状（可含位图）| `Shape.CreateDropShadow` 等 |
| **④ 透明度**（Transparency） | 任意形状 | `Shape.Transparency.Apply*` |

## 一、位图效果插件（ApplyBitmapEffect）

### 1.1 方法签名

```vba
' 位图形状的效果入口：Shape.Bitmap（Bitmap 对象）
Sub Bitmap.ApplyBitmapEffect(UndoString As String, Command As String)
```

- `UndoString`：撤销历史列表中显示的操作名（自定义字符串）
- `Command`：插件名 + 参数列表，格式为 `"EffectName Param1=Val1,Param2=Val2,..."`

> ⚠️ 前提条件：必须先确保形状是 `cdrBitmapShape`，且颜色模式为 RGB（`cdrRGBColorImage`）。

```vba
' 标准安全调用模板
Dim s As Shape
Set s = ActiveShape
If s.Type = cdrBitmapShape Then
    If s.Bitmap.Mode <> cdrRGBColorImage Then
        s.Bitmap.ConvertTo cdrRGBColorImage  ' 先转 RGB
    End If
    s.Bitmap.ApplyBitmapEffect "效果名", "EffectPluginName Param1=Val1,..."
End If
```

### 1.2 已知效果插件及参数列表

效果名（Command 字符串格式）需要通过**宏录制器**确认精确拼写。以下参数来源于官方示例及社区验证：

#### 模糊类（Blur）

| 效果 | Command 字符串 | 主要参数 |
|------|--------------|---------|
| 高斯模糊 | `"GaussianBlurEffect Radius=10"` | `Radius`（0–250，像素） |
| 运动模糊 | `"MotionBlurEffect Angle=45,Distance=20"` | `Angle`（0–360°）、`Distance`（像素） |
| 缩放模糊 | `"ZoomBlurEffect Amount=60"` | `Amount`（0–100） |
| 柔和焦点 | `"SoftFocusEffect Amount=40,Edge=20"` | `Amount`、`Edge` |
| 智能模糊 | `"SmartBlurEffect Radius=10,Threshold=15"` | `Radius`、`Threshold` |
| 涡流（Whirlpool） | `"WhirlpoolEffect WhirpoolSpacing=20,WhirlpoolSmear=9,WhirlpoolTwist=70,WhirlpoolStreak=60,WhirpoolWarp=1,WhirlpoolRandomSeed=0"` | 6 个参数 |

> `WhirlpoolEffect` 是唯一在官方 X7 手册中有完整示例的插件名（第 1 册 7685 页）。

#### 锐化类（Sharpen）

| 效果 | Command 字符串 | 主要参数 |
|------|--------------|---------|
| 锐化 | `"SharpenEffect Amount=50"` | `Amount`（0–100） |
| 方向锐化 | `"DirectionalSharpenEffect Angle=90,Strength=50"` | `Angle`、`Strength` |
| 自适应去除杂点 | `"AdaptiveUnsharpEffect Amount=50,Radius=5"` | `Amount`、`Radius` |

#### 噪点类（Noise）

| 效果 | Command 字符串 | 主要参数 |
|------|--------------|---------|
| 添加噪点 | `"AddNoiseEffect Amount=20,Type=1"` | `Amount`（0–100）、`Type`（0=均匀，1=高斯） |
| 去斑 | `"DespeckleEffect"` | 无参数 |

#### 创意类（Creative）

| 效果 | Command 字符串 | 主要参数 |
|------|--------------|---------|
| 浮雕 | `"EmbossEffect Amount=50,Angle=135"` | `Amount`（0–100）、`Angle`（0–360°） |
| 曝光过度（Solarize） | `"SolarizeEffect Level=50"` | `Level`（0–255） |

> ⚠️ **重要说明**：`ApplyBitmapEffect` 的 Command 字符串均为 **CorelDRAW 内部插件 API 调用**，不在 COM 对象模型的公开枚举中。官方文档只提供了 WhirlpoolEffect 一个示例。其余效果名及参数需要通过 **宏录制器**（`工具 → 宏 → 宏录制器`）在 CorelDRAW 中手动操作后查看生成代码来确认。另外，CorelDRAW 2019 及更高版本存在该方法参数无效的已知 Bug，仅 X7 稳定可用。

### 1.3 完整示例：对所有页面位图应用高斯模糊

```vba
Sub ApplyGaussianBlurToAllBitmaps()
    Dim s As Shape
    For Each s In ActivePage.Shapes
        If s.Type = cdrBitmapShape Then
            If s.Bitmap.Mode <> cdrRGBColorImage Then
                s.Bitmap.ConvertTo cdrRGBColorImage
            End If
            s.Bitmap.ApplyBitmapEffect "Gaussian Blur", "GaussianBlurEffect Radius=10"
        End If
    Next s
End Sub
```

## 二、色调 / 颜色调整（Shape.ApplyEffect*）

这 6 个方法可作用于**任意形状（含位图）**，直接在 `Shape` 对象上调用，无需进入 `Bitmap` 子对象。

### 2.1 方法列表

| 方法 | 作用 | 主要参数范围 |
|------|------|------------|
| `ApplyEffectBCI` | 亮度/对比度/强度 | 各 -100 ~ 100 |
| `ApplyEffectColorBalance` | 色彩平衡（CMY 通道） | 各 -100 ~ 100，三个色调区域 |
| `ApplyEffectGamma` | Gamma 校正 | 0.01 ~ 10.0 |
| `ApplyEffectHSL` | 色相/饱和度/亮度 | H: -180~180, S/L: -100~100 |
| `ApplyEffectInvert` | 反转颜色 | 无参数 |
| `ApplyEffectPosterize` | 色调分离（减色） | Level 2~8 |

### 2.2 方法签名

```vba
Sub   Shape.ApplyEffectBCI(Brightness As Long, Contrast As Long, Intensity As Long)

Sub   Shape.ApplyEffectColorBalance(CyanRed As Long, MagentaGreen As Long, YellowBlue As Long,
          [ApplyToShadows As Boolean = True], [ApplyToMidtones As Boolean = True],
          [ApplyToHighlights As Boolean = True], [PreserveLuminance As Boolean = True])

Sub   Shape.ApplyEffectGamma(Gamma As Double)

Sub   Shape.ApplyEffectHSL(Hue As Variant, Saturation As Variant, Lightness As Variant)

Sub   Shape.ApplyEffectInvert()

Sub   Shape.ApplyEffectPosterize(Level As Long)
```

### 2.3 代码示例

```vba
Sub ColorAdjustDemo()
    Dim s As Shape
    Set s = ActiveShape

    ' 亮度 +20，对比度 +30，强度 0
    s.ApplyEffectBCI 20, 30, 0

    ' 色彩平衡：偏红 +20，偏绿 -10，保持蓝色，仅作用于中间调
    s.ApplyEffectColorBalance 20, -10, 0, False, True, False, True

    ' Gamma 校正为 1.5（提亮中间调）
    s.ApplyEffectGamma 1.5

    ' HSL：色相 +30°，饱和度 -20，亮度不变
    s.ApplyEffectHSL 30, -20, 0

    ' 反转颜色
    s.ApplyEffectInvert

    ' 色调分离：4 色阶
    s.ApplyEffectPosterize 4
End Sub
```

> 这些方法也可批量应用于 `ShapeRange`：
> ```vba
> ActivePage.Shapes.All.CreateSelection
> ActiveSelection.ApplyEffectBCI 10, 20, 0
> ```

## 三、CorelDRAW 矢量特效（Shape.Create*）

这类特效既可用于矢量图形也可用于位图，产生 **`Effect`** 对象，通过 `Shape.Effects` 集合管理。

### 3.1 效果类型（cdrEffectType 枚举）

|-----|---|---------|----------------|
| `cdrBlend` | 0 | 调和 | `Effect.Blend`（`EffectBlend`） |
| `cdrExtrude` | 1 | 立体化 | `Effect.Extrude`（`EffectExtrude`） |
| `cdrEnvelope` | 2 | 封套 | `Effect.Envelope`（`EffectEnvelope`） |
| `cdrTextOnPath` | 3 | 文本沿路径 | `Effect.TextOnPath` |
| `cdrControlPath` | 4 | 控制路径 | `Effect.ControlPath` |
| `cdrDropShadow` | 5 | 阴影 | `Effect.DropShadow`（`EffectDropShadow`） |
| `cdrContour` | 6 | 轮廓图 | `Effect.Contour`（`EffectContour`） |
| `cdrDistortion` | 7 | 变形 | `Effect.Distortion`（`EffectDistortion`） |
| `cdrPerspective` | 8 | 透视 | `Effect.Perspective`（`EffectPerspective`） |
| `cdrLens` | 9 | 透镜 | `Effect.Lens`（`EffectLens`） |
| `cdrCustomEffect` | 10 | 自定义特效 | `Effect.Custom`（`CustomEffect`） |

### 3.2 阴影（DropShadow）

#### 创建方法

```vba
Function Shape.CreateDropShadow(
    [Type As cdrDropShadowType = cdrDropShadowFlat],   ' 阴影类型
    [Opacity As Long = 50],          ' 不透明度 0~100
    [Feather As Long = 15],          ' 羽化值 0~100
    [OffsetX As Double],             ' X 偏移（文档单位）
    [OffsetY As Double],             ' Y 偏移
    [Color As Color],                ' 阴影颜色
    [FeatherType As cdrFeatherType], ' 羽化方式
    [FeatherEdge As cdrEdgeType],    ' 边缘类型
    [PerspectiveAngle As Double = -45],
    [PerspectiveStretch As Double = 1],
    [Fade As Long = 1],              ' 消退值（透视阴影）
    [MergeMode As cdrMergeMode]      ' 混合模式
) As Effect
```

#### cdrDropShadowType 枚举

| 常量 | 值 | 说明 |
|-----|---|------|
| `cdrDropShadowFlat` | 0 | 平面阴影（偏移） |
| `cdrDropShadowBottom` | 1 | 底部透视 |
| `cdrDropShadowTop` | 2 | 顶部透视 |
| `cdrDropShadowLeft` | 3 | 左侧透视 |
| `cdrDropShadowRight` | 4 | 右侧透视 |

#### 代码示例

```vba
Sub DropShadowDemo()
    Dim s As Shape
    Set s = ActiveLayer.CreateRectangle(0, 0, 5, 3)
    s.Fill.UniformColor.RGBAssign 0, 100, 200

    ' 创建平面阴影：50% 不透明，15 羽化，偏移 0.3/-0.3
    Dim eff As Effect
    Set eff = s.CreateDropShadow(cdrDropShadowFlat, 50, 15, 0.3, -0.3, _
                                 CreateRGBColor(0, 0, 0))

    ' 事后修改阴影属性
    eff.DropShadow.Color.RGBAssign 100, 0, 0   ' 改为红色阴影
    eff.DropShadow.Feather = 30
    eff.DropShadow.Opacity = 80
End Sub
```

#### 遍历并修改已有阴影

```vba
Sub ModifyAllDropShadows()
    Dim s As Shape, e As Effect
    For Each s In ActivePage.Shapes
        For Each e In s.Effects
            If e.Type = cdrDropShadow Then
                e.DropShadow.Color.RGBAssign 255, 0, 0  ' 全部改为红色
                e.DropShadow.Feather = 30
            End If
        Next e
    Next s
End Sub
```

```vba
Sub LensDemo()
    ' 在页面中心创建椭圆鱼眼镜头（100% 强度，冻结）
    Dim s As Shape, e As Effect
    Dim cx As Double, cy As Double
    cx = ActivePage.SizeWidth / 2
    cy = ActivePage.SizeHeight / 2
    Set s = ActiveLayer.CreateEllipse2(cx, cy, 2)
    Set e = s.CreateLens(cdrLensFishEye, 100)
    e.Lens.Freeze   ' 冻结：使透镜效果独立于下方对象

    Set s = ActiveLayer.CreateEllipse2(cx, cy, 1.5)
    s.CreateLens cdrLensBrighten, 50
End Sub
```

## 四、透明度（Transparency）

`Shape.Transparency` 属性返回 `Transparency` 对象，可为任意形状添加渐变/纯色/图案透明度。

### 4.1 方法列表

| 方法 | 说明 |
|------|------|
| `ApplyUniformTransparency(Value)` | 均匀透明度，0~100 |
| `ApplyFountainTransparency(Start, End, Type, ...)` | 渐变透明度 |
| `ApplyPatternTransparency(Type, FileName, ...)` | 图案透明度 |
| `ApplyTextureTransparency(TextureName, LibraryName)` | 纹理透明度 |
| `ApplyNoTransparency()` | 移除透明度 |

### 4.2 AppliedTo 属性

```vba
' 控制透明度作用于填充还是轮廓
s.Transparency.AppliedTo = cdrApplyToFill     ' 仅填充（默认）
s.Transparency.AppliedTo = cdrApplyToOutline  ' 仅轮廓
s.Transparency.AppliedTo = cdrApplyToAll      ' 全部
```

### 4.3 代码示例

```vba
Sub TransparencyDemo()
    Dim s As Shape
    Set s = ActiveLayer.CreateRectangle(0, 0, 5, 3)
    s.Fill.ApplyFountainFill CreateRGBColor(255,0,0), CreateRGBColor(255,255,0)
    s.Outline.Width = 0.3

    ' 均匀 50% 透明
    s.Transparency.ApplyUniformTransparency 50

    ' 线性渐变透明度（左 0% → 右 90%）
    ' s.Transparency.ApplyFountainTransparency 0, 90, cdrLinearFountainFill

    ' 仅对轮廓应用
    ' s.Transparency.AppliedTo = cdrApplyToOutline

    ' 纹理透明度
    ' s.Transparency.ApplyTextureTransparency "Cosmic Energy", ""

    ' 移除
    ' s.Transparency.ApplyNoTransparency
End Sub
```

## 五、位图基础操作（Bitmap 对象）

位图的基础属性和操作也会影响视觉效果：

### 5.1 颜色模式转换

```vba
Dim s As Shape
Set s = ActiveShape

' 转 RGB（大多数效果的前提条件）
s.Bitmap.ConvertTo cdrRGBColorImage

' 转灰度
s.Bitmap.ConvertTo cdrGrayscaleImage

' 转 CMYK
s.Bitmap.ConvertTo cdrCMYKColorImage

' 转黑白（含渲染类型选择）
s.Bitmap.ConvertToBW cdrRenderLineArt, , 97     ' 线条艺术，阈值 97
s.Bitmap.ConvertToBW cdrRenderOrderedDither      ' 有序抖动
```

#### cdrImageMode 枚举

| 常量 | 说明 |
|------|------|
| `cdrImageBlackWhite` (0) | 黑白 |
| `cdrImageGrayscale` (2) | 灰度 |
| `cdrImagePaletted` (3) | 调色板 |
| `cdrImageRGB` (4) | RGB |
| `cdrImageCMYK` (5) | CMYK |
| `cdrImageDuotone` (6) | 双色调 |
| `cdrImageLAB` (7) | LAB |

### 5.2 重采样与裁剪

```vba
' 重采样到 300x300 像素（保持最大尺寸比例）
s.Bitmap.Resample 300, 300, True     ' Width, Height, AntiAlias

' 按分辨率重采样
s.Bitmap.Resample , , True, 300, 300  ' ResolutionX=300, ResolutionY=300

' 扩展位图画布（每边各加 20 像素）
s.Bitmap.Inflate 20, 20

' 裁剪到裁切路径
If s.Bitmap.Cropped Then s.Bitmap.Crop
```

### 5.3 矢量形状转位图

```vba
' 推荐：ConvertToBitmapEx（返回新的 Shape）
Dim bmpShape As Shape
Set bmpShape = ActiveShape.ConvertToBitmapEx( _
    cdrRGBColorImage, _   ' 颜色模式
    False,                ' 不抖动
    True,                 ' 透明背景
    300,                  ' 分辨率 300 dpi
    cdrNormalAntiAliasing)

' 然后对位图应用效果
bmpShape.Bitmap.ApplyBitmapEffect "Gaussian Blur", "GaussianBlurEffect Radius=5"
```

## 六、PowerClip（剪贴蒙版）

PowerClip 将一个形状嵌入另一个形状的轮廓内，常用于为位图创建异形裁切框架：

```vba
Sub PowerClipDemo()
    ' 创建椭圆容器 + 将位图放入其中
    Dim container As Shape, bmp As Shape
    Set container = ActiveLayer.CreateEllipse(2, 2, 8, 6)
    container.Fill.UniformColor.RGBAssign 200, 200, 200

    ' bmp 是页面上已有的位图
    Set bmp = ActivePage.FindShape(, cdrBitmapShape)
    If Not bmp Is Nothing Then
        bmp.AddToPowerClip container          ' 将位图放入椭圆内
        container.PowerClip.ContentsLocked = True  ' 锁定内容
    End If
End Sub

Sub EditPowerClip()
    Dim s As Shape
    Set s = ActiveShape   ' 选中 PowerClip 容器
    s.PowerClip.EnterEditMode    ' 进入编辑模式
    ' ... 对内部形状做修改 ...
    s.PowerClip.LeaveEditMode    ' 退出编辑模式
End Sub
```

## 七、综合示例：从形状到带效果的位图

```vba
Sub BitmapEffectsFullDemo()
    If Application.Documents.Count = 0 Then CreateDocument

    ' 1. 创建矩形并设置填充
    Dim s As Shape
    Set s = ActiveLayer.CreateRectangle(0, 0, 6, 4)
    s.Fill.ApplyFountainFill CreateRGBColor(255,0,0), CreateRGBColor(0,0,255), _
        , 45

    ' 2. 添加阴影
    s.CreateDropShadow cdrDropShadowFlat, 60, 20, 0.3, -0.3, CreateRGBColor(0,0,0)

    ' 3. 转为位图（300 dpi，RGB）
    Dim bmp As Shape
    Set bmp = s.ConvertToBitmapEx(cdrRGBColorImage, False, True, 300)

    ' 4. 应用颜色调整
    bmp.ApplyEffectBCI 10, 20, 0            ' 提亮 + 对比度
    bmp.ApplyEffectHSL 15, -10, 5           ' 略微偏暖，降饱和

    ' 5. 应用插件效果（高斯模糊）
    bmp.Bitmap.ApplyBitmapEffect "Gaussian Blur", "GaussianBlurEffect Radius=3"

    ' 6. 添加透明度
    bmp.Transparency.ApplyFountainTransparency 0, 70, cdrLinearFountainFill

    MsgBox "位图效果演示完成！"
End Sub
```
