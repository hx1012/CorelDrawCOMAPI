## 一、对象模型总览

```
Application
├── Printers          → SystemPrinters（已安装打印机集合）
│   └── [n]          → Printer（单台打印机）
└── PrintJob          → PrintJob（全局打印任务）
    ├── Documents     → PrintDocuments
    ├── Pages         → PrintPages
    └── Settings      → PrintSettings

Document
├── PrintSettings     → PrintSettings（文档打印设置，核心入口）
│   ├── Printer       → Printer
│   ├── Options       → PrintOptions（常规选项）
│   ├── Layout        → PrintLayout（页面布局）
│   ├── PostScript    → PrintPostScript（PostScript 设置）
│   ├── Prepress      → PrintPrepress（印前处理）
│   ├── Separations   → PrintSeparations（色彩分色）
│   │   └── Plates    → SeparationPlates → SeparationPlate
│   └── Trapping      → PrintTrapping（陷印）
└── PrintOut()        → 执行打印
```

**最常用的打印入口**：
```vba
' 方式 1：通过文档打印（最常用）
ActiveDocument.PrintOut

' 方式 2：通过 PrintSettings 打印（不重新分析内容，速度更快）
ActiveDocument.PrintSettings.PrintOut

' 方式 3：通过全局 PrintJob 打印（多文档批量打印）
Application.PrintJob.PrintOut
```

## 二、系统打印机（SystemPrinters / Printer）

### 2.1 SystemPrinters 集合

`Application.Printers`（简写为 `Printers`）返回 `SystemPrinters` 对象，代表系统中所有已安装的打印机。

| 属性 | 类型 | 说明 |
|------|------|------|
| `Count` | `Long`（只读）| 已安装打印机的数量 |
| `Default` | `Printer`（只读）| 默认打印机 |
| `Item(nIndex)` | `Printer`（只读）| 按索引返回打印机（默认属性，可省略 `.Item`）|

```vba
' 显示默认打印机名称和端口
Sub ShowDefaultPrinter()
    If Printers.Count = 0 Then
        MsgBox "系统中未安装打印机。"
        Exit Sub
    End If
    MsgBox "默认打印机：" & Printers.Default.Name & _
           "  端口：" & Printers.Default.Port
End Sub

' 列出所有已安装打印机
Sub ListAllPrinters()
    Dim s As String
    Dim i As Integer
    For i = 1 To Printers.Count
        s = s & Printers(i).Name & "  [" & Printers(i).Port & "]" & vbCr
    Next i
    MsgBox "系统已安装 " & Printers.Count & " 台打印机：" & vbCr & s
End Sub
```

### 2.2 Printer 对象属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `Name` | `String`（只读）| 打印机名称 |
| `Type` | `String`（只读）| 打印机型号名称 |
| `Description` | `String`（只读）| 打印机描述 |
| `Port` | `String`（只读）| 端口名称（如 `LPT1`、`USB001`）|
| `Default` | `Boolean`（只读）| 是否为默认打印机 |
| `ColorEnabled` | `Boolean`（只读）| 是否支持彩色打印 |
| `PostScriptEnabled` | `Boolean`（只读）| 是否支持 PostScript |
| `PageSizeMatchingSupported` | `Boolean`（只读）| 是否支持页面尺寸匹配 |
| `Ready` | `Boolean`（只读）| 打印机是否就绪 |

### 2.3 Printer 方法

| 方法 | 说明 |
|------|------|
| `ShowDialog()` | 显示打印机属性对话框 |

```vba
' 查找默认打印机并显示其属性对话框
Sub ShowDefaultPrinterDialog()
    Printers.Default.ShowDialog
End Sub

' 列出所有彩色 PostScript 打印机
Sub ListColorPSPrinters()
    Dim prn As Printer
    Dim s As String
    Dim i As Integer
    For i = 1 To Printers.Count
        Set prn = Printers(i)
        If prn.ColorEnabled And prn.PostScriptEnabled Then
            s = s & prn.Type & " (" & prn.Port & ")" & vbCr
        End If
    Next i
    If s = "" Then
        MsgBox "未找到彩色 PostScript 打印机。"
    Else
        MsgBox "彩色 PostScript 打印机：" & vbCr & s
    End If
End Sub

' 查找是否安装了特定型号的打印机
Sub FindSpecificPrinter()
    Dim prn As Printer
    Dim i As Integer
    For i = 1 To Printers.Count
        Set prn = Printers(i)
        If prn.Type = "HP LaserJet 5P/5MP PostScript" Then
            MsgBox prn.Type & " 已安装。"
            Exit For
        End If
    Next i
End Sub
```

## 三、打印设置（PrintSettings）

`Document.PrintSettings` 是整个打印系统的**核心入口**，它返回 `PrintSettings` 对象，统一管理所有打印选项。

### 3.1 PrintSettings 属性一览

| 属性 | 类型 | 说明 |
|------|------|------|
| `Printer` | `Printer` | 当前选用的打印机（可读写）|
| `Copies` | `Long` | 打印份数 |
| `Collate` | `Boolean` | 是否逐份打印（多份时页面在一起）|
| `PrintRange` | `PrnPrintRange` | 打印范围（全文档/当前页/选区/页面范围）|
| `PageRange` | `String` | 页面范围字符串（如 `"1,3,5-8"`），需配合 `prnPageRange`|
| `PageSet` | `PrnPageSet` | 打印页面子集（全部/奇数/偶数/左侧/右侧）|
| `PageMatchingMode` | `PrnPageMatchingMode` | 页面匹配模式 |
| `PaperSize` | `PrnPaperSize` | 纸张尺寸（预设）|
| `PaperOrientation` | `PrnPaperOrientation` | 纸张方向 |
| `PaperWidth` | `Double`（只读）| 纸张宽度（文档单位）|
| `PaperHeight` | `Double`（只读）| 纸张高度（文档单位）|
| `PrintToFile` | `Boolean` | 是否打印到文件 |
| `FileName` | `String` | 打印到文件时的文件名 |
| `FileMode` | `PrnFileMode` | 文件输出模式（单文件/按页/按分色版）|
| `ForMac` | `Boolean` | 打印到文件时是否为 Mac 可读格式 |
| `UsePPD` | `Boolean` | 是否使用 PPD 文件 |
| `PPDFile` | `String` | PPD 文件路径（配合 `UsePPD = True`）|
| `Options` | `PrintOptions`（只读）| 常规打印选项 |
| `Layout` | `PrintLayout`（只读）| 页面布局设置 |
| `PostScript` | `PrintPostScript`（只读）| PostScript 设置 |
| `Prepress` | `PrintPrepress`（只读）| 印前处理设置 |
| `Separations` | `PrintSeparations`（只读）| 色彩分色设置 |
| `Trapping` | `PrintTrapping`（只读）| 陷印设置 |

### 3.2 PrintSettings 方法一览

| 方法 | 说明 |
|------|------|
| `PrintOut()` | 不重新分析内容直接打印 |
| `PrintColorProof(ProofSettings)` | 打印颜色打样 |
| `ShowDialog()` | 显示"打印"对话框，返回 `Boolean`（用户是否确认）|
| `Reset()` | 恢复为默认打印样式 |
| `Save(FileName)` | 保存当前打印样式到文件（`.prs`）|
| `Load(FileName)` | 从文件加载已保存的打印样式 |
| `SelectPrinter(Name)` | 按名称选择打印机 |
| `SetPaperSize(PaperSize, Orientation)` | 设置预设纸张尺寸和方向 |
| `SetCustomPaperSize(Width, Height, Orientation)` | 设置自定义纸张尺寸 |

### 3.3 基本打印示例

```vba
' 最简单的打印：打印活动文档
Sub SimplePrint()
    ActiveDocument.PrintOut
End Sub

' 打印多份并逐份排序
Sub PrintMultipleCopies()
    With ActiveDocument.PrintSettings
        .Copies = 3
        .Collate = True
    End With
    ActiveDocument.PrintOut
End Sub

' 只打印第 2、4 页
Sub PrintSpecificPages()
    With ActiveDocument.PrintSettings
        .PrintRange = prnPageRange
        .PageRange = "2,4"
    End With
    ActiveDocument.PrintOut
End Sub

' 只打印当前页
Sub PrintCurrentPage()
    ActiveDocument.PrintSettings.PrintRange = prnCurrentPage
    ActiveDocument.PrintOut
End Sub

' 只打印奇数页
Sub PrintOddPages()
    With ActiveDocument.PrintSettings
        .PrintRange = prnWholeDocument
        .PageSet = prnPageSetOdd
    End With
    ActiveDocument.PrintOut
End Sub

' 选择打印机并打印
Sub SelectAndPrint()
    With ActiveDocument.PrintSettings
        .SelectPrinter "Microsoft Print to PDF"
        .Copies = 1
    End With
    ActiveDocument.PrintOut
End Sub

' 打印到 .prn 文件
Sub PrintToFile()
    With ActiveDocument.PrintSettings
        .PrintToFile = True
        .ForMac = False
        .FileMode = prnSingleFile
        .FileName = "C:\Output\MyPrint.prn"
    End With
    ActiveDocument.PrintOut
End Sub

' 保存并加载打印样式
Sub SaveAndLoadPrintStyle()
    With ActiveDocument.PrintSettings
        .Copies = 2
        .Separations.Enabled = True
        .Save "C:\PrintStyles\MySepStyle.prs"
        .Reset
    End With
    ActiveDocument.PrintOut          ' 用默认样式打印一次
    ActiveDocument.PrintSettings.Load "C:\PrintStyles\MySepStyle.prs"
    ActiveDocument.PrintOut          ' 用保存的样式再打印一次
End Sub

' 设置纸张尺寸为 A4 横向
Sub SetA4Landscape()
    ActiveDocument.PrintSettings.SetPaperSize prnPaperSizeA4, prnPaperLandscape
End Sub

' 设置自定义纸张尺寸（单位：文档单位，默认 mm）
Sub SetCustomPaper()
    ' 420 × 594 mm（A2 大小），纵向
    ActiveDocument.PrintSettings.SetCustomPaperSize 420, 594, prnPaperPortrait
End Sub

' 显示打印对话框，然后批量打印文件夹中所有 CDR 文件
Sub BatchPrintWithDialog()
    Dim folder As String
    Dim fileName As String
    Dim doc As Document
    Dim prn As CorelDRAW.Printer
    Dim nCopies As Long
    Dim bFirst As Boolean
    folder = "C:\MyDesigns\"
    fileName = Dir(folder & "*.cdr")
    bFirst = True
    Do While fileName <> ""
        Set doc = OpenDocument(folder & fileName)
        With doc.PrintSettings
            If bFirst Then
                If Not .ShowDialog Then Exit Sub   ' 用户取消
                Set prn = .Printer
                nCopies = .Copies
                bFirst = False
            Else
                Set .Printer = prn
                .Copies = nCopies
            End If
        End With
        doc.PrintOut
        doc.Close
        fileName = Dir()
    Loop
End Sub
```

## 四、打印选项（PrintOptions）

通过 `PrintSettings.Options` 访问，控制要打印哪些内容以及如何处理颜色、位图、字体等。

### 4.1 PrintOptions 属性一览

| 属性 | 类型 | 说明 |
|------|------|------|
| `ColorMode` | `PrnColorMode` | 颜色模式（全彩/灰度/黑白）|
| `ObjectsColorMode` | `PrnObjectsColorMode` | 对象颜色模式（CMYK/RGB/灰度/原始）|
| `PrintBitmaps` | `Boolean` | 是否打印位图 |
| `PrintVectors` | `Boolean` | 是否打印矢量图 |
| `PrintText` | `Boolean` | 是否打印文字 |
| `TextInBlack` | `Boolean` | 是否将所有文字打印为黑色 |
| `RasterizePage` | `Boolean` | 是否将整页栅格化后打印 |
| `RasterizeResolution` | `Long` | 栅格化打印的分辨率（dpi）|
| `FountainSteps` | `Long` | 渐变填充步数（分辨率）|
| `ColorResolution` | `Long` | 彩色位图打印分辨率（dpi）|
| `GrayResolution` | `Long` | 灰度位图打印分辨率（dpi）|
| `MonoResolution` | `Long` | 单色位图打印分辨率（dpi）|
| `DownsampleColor` | `Boolean` | 是否降采样彩色位图 |
| `DownsampleGray` | `Boolean` | 是否降采样灰度位图 |
| `DownsampleMono` | `Boolean` | 是否降采样单色位图 |
| `MarksToPage` | `Boolean` | 印前标记放置在页面内（而非纸张边缘）|
| `UseColorProfile` | `Boolean` | 是否使用颜色配置文件 |
| `JobInformation` | `Boolean` | 是否打印作业信息页 |
| `AppInfo` | `Boolean` | 作业信息页：包含应用程序信息 |
| `DriverInfo` | `Boolean` | 作业信息页：包含驱动程序信息 |
| `FontInfo` | `Boolean` | 作业信息页：包含字体信息 |
| `PrintJobInfo` | `Boolean` | 作业信息页：包含打印作业信息 |
| `SepsInfo` | `Boolean` | 作业信息页：包含分色信息 |
| `LinkInfo` | `Boolean` | 作业信息页：包含链接信息 |
| `InRIPTrapInfo` | `Boolean` | 是否使用 In-RIP 陷印 |

```vba
' 仅打印矢量，不打印位图（快速出图）
Sub PrintVectorsOnly()
    With ActiveDocument.PrintSettings.Options
        .PrintVectors = True
        .PrintBitmaps = False
        .PrintText = True
    End With
    ActiveDocument.PrintOut
End Sub

' 将整页栅格化后打印（解决复杂页面打印问题）
Sub RasterizePrint()
    With ActiveDocument.PrintSettings.Options
        .RasterizePage = True
        .RasterizeResolution = 300
    End With
    ActiveDocument.PrintOut
End Sub

' 灰度打印，并降采样所有位图
Sub GrayscalePrint()
    With ActiveDocument.PrintSettings.Options
        .ColorMode = prnModeGrayscale
        .DownsampleColor = True
        .DownsampleGray = True
        .DownsampleMono = True
        .ColorResolution = 150
        .GrayResolution = 150
        .MonoResolution = 600
    End With
    ActiveDocument.PrintOut
End Sub

' 使用颜色配置文件打印
Sub PrintWithColorProfile()
    ActiveDocument.PrintSettings.Options.UseColorProfile = True
    ActiveDocument.PrintOut
End Sub

' 打印作业信息页
Sub PrintWithJobInfo()
    With ActiveDocument.PrintSettings.Options
        .JobInformation = True
        .AppInfo = True
        .DriverInfo = True
        .FontInfo = True
        .PrintJobInfo = True
        .SepsInfo = False
    End With
    ActiveDocument.PrintOut
End Sub
```

## 五、页面布局（PrintLayout）

通过 `PrintSettings.Layout` 访问，控制文档页面在打印纸上的放置方式。

### 5.1 PrintLayout 属性一览

| 属性 | 类型 | 说明 |
|------|------|------|
| `Placement` | `PrnPlaceType` | 页面在纸上的放置位置 |
| `PrintTiledPages` | `Boolean` | 是否平铺打印（大页面跨多张纸）|
| `PrintTilingMarks` | `Boolean` | 平铺时是否打印拼接标记 |
| `TileOverlap` | `Double` | 平铺页面之间的重叠量 |
| `UseBleedLimit` | `Boolean` | 是否使用出血值 |
| `BleedLimit` | `Double` | 出血量（文档单位）|

```vba
' 居中打印，带出血
Sub PrintCenteredWithBleed()
    With ActiveDocument.PrintSettings.Layout
        .Placement = prnPlaceCenterPage
        .UseBleedLimit = True
        .BleedLimit = 3           ' 3 mm 出血
    End With
    ActiveDocument.PrintOut
End Sub

' 将大文档平铺到多张 A4 纸上
Sub TilePrint()
    With ActiveDocument.PrintSettings
        .SetPaperSize prnPaperSizeA4, prnPaperPortrait
        With .Layout
            .Placement = prnPlaceAsInDocument
            .PrintTiledPages = True
            .PrintTilingMarks = True
            .TileOverlap = 10    ' 10 mm 重叠
        End With
    End With
    ActiveDocument.PrintOut
End Sub

' 缩放适合页面
Sub FitToPage()
    ActiveDocument.PrintSettings.Layout.Placement = prnPlaceFitToPage
    ActiveDocument.PrintOut
End Sub
```

## 六、PostScript 设置（PrintPostScript）

通过 `PrintSettings.PostScript` 访问，控制 PostScript 输出的技术细节。仅在打印到 PostScript 设备时生效。

### 6.1 PrintPostScript 属性一览

| 属性 | 类型 | 说明 |
|------|------|------|
| `Level` | `PrnPostScriptLevel` | PostScript 级别（1/2/3）|
| `AutoIncreaseFlatness` | `Boolean` | 自动增大曲线平坦度（避免打印错误）|
| `Flatness` | `Long` | 曲线平坦度值 |
| `AutoIncreaseFountainSteps` | `Boolean` | 自动增加渐变步数（减少条带）|
| `OptimizeFountainFills` | `Boolean` | 优化渐变填充（必要时减少步数）|
| `FountainSteps`* | 通过 `PrintOptions.FountainSteps` 设置 | 渐变步数 |
| `MaxPointsPerCurve` | `Long` | 曲线每段最大控制点数 |
| `ScreenFrequency` | `Long` | 基本半调网频（lpi）|
| `ConformToDSC` | `Boolean` | 是否符合 DSC 文档结构约定 |
| `JPEGCompression` | `Boolean` | 是否对打印作业中的位图进行 JPEG 压缩 |
| `JPEGQuality` | `Long` | JPEG 压缩质量（需 `JPEGCompression = True`）|
| `DownloadType1` | `Boolean` | 是否下载 Type 1 字体到打印机 |
| `TrueTypeToType1` | `Boolean` | 是否将 TrueType 字体转换为 Type 1 后下载 |
| `MaintainOPILinks` | `Boolean` | 蒸馏 PS 文件时是否维护 OPI 链接 |
| `ResolveDCSLinks` | `Boolean` | 是否解析 DCS 链接 |
| `pdfBookmarks` | `Boolean` | 蒸馏为 PDF 时是否包含书签 |
| `PDFHyperlinks` | `Boolean` | 蒸馏为 PDF 时是否包含超链接 |
| `PDFStartup` | `PrnPDFStartup` | PDF 打开时的默认显示方式 |

```vba
' 设置 PostScript Level 3，并优化渐变和曲线
Sub SetupPS3Print()
    With ActiveDocument.PrintSettings.PostScript
        .Level = prnPSLevel3
        .AutoIncreaseFlatness = True
        .AutoIncreaseFountainSteps = True
        .OptimizeFountainFills = True
        .ScreenFrequency = 150      ' 150 lpi
    End With
    ActiveDocument.PrintOut
End Sub

' 下载字体到 PostScript 打印机
Sub PrintWithFontDownload()
    With ActiveDocument.PrintSettings.PostScript
        .DownloadType1 = True
        .TrueTypeToType1 = True
    End With
    ActiveDocument.PrintOut
End Sub

' 打印到 PS 文件，保留 OPI 链接
Sub PrintToPS()
    With ActiveDocument
        .PrintSettings.PostScript.MaintainOPILinks = True
        .PrintSettings.PostScript.ResolveDCSLinks = True
        .PrintSettings.PrintToFile = True
        .PrintSettings.FileName = "C:\Output\MyDoc.ps"
        .PrintOut
    End With
End Sub

' 设置蒸馏为 PDF 时的选项
Sub SetPDFDistillOptions()
    With ActiveDocument.PrintSettings.PostScript
        .PDFStartup = prnPDFOutlines    ' 打开时显示文档大纲
        .pdfBookmarks = True
        .PDFHyperlinks = True
    End With
End Sub

' 使用 JPEG 压缩打印位图（减小文件体积）
Sub PrintWithJPEGCompression()
    With ActiveDocument.PrintSettings.PostScript
        .JPEGCompression = True
        .JPEGQuality = 80           ' 0–100，值越大质量越高
    End With
    ActiveDocument.PrintOut
End Sub
```

## 七、印前处理（PrintPrepress）

通过 `PrintSettings.Prepress` 访问，控制专业印刷所需的印前标记和信息。

### 7.1 PrintPrepress 属性一览

| 属性 | 类型 | 说明 |
|------|------|------|
| `CropMarks` | `Boolean` | 是否打印裁切/折叠标记（页面内）|
| `ExteriorCropMarks` | `Boolean` | 是否打印页面外的裁切标记 |
| `RegistrationMarks` | `Boolean` | 是否打印套准标记 |
| `RegistrationStyle` | `PrnRegistrationStyle` | 套准标记样式 |
| `ColorCalibrationBar` | `Boolean` | 是否打印颜色校准条 |
| `DensitometerScale` | `Boolean` | 是否打印密度计刻度 |
| `Densities(Index)` | `Long` | 密度计刻度各方块的密度值 |

```vba
    MsgBox "已打印 " & job.Documents.Count & " 个文档。"
```

### 示例 1：印刷级输出（CMYK 分色 + PostScript + 印前标记）

```vba
Sub ProfessionalPressOutput()
    Dim doc As Document
    Dim s As Shape
    Set doc = CreateDocument

    ' 创建测试图形
    Set s = doc.ActiveLayer.CreateRectangle(10, 200, 100, 100)
    s.Fill.UniformColor.CMYKAssign 100, 0, 0, 0
    Set s = doc.ActiveLayer.CreateRectangle(110, 200, 200, 100)
    s.Fill.UniformColor.CMYKAssign 0, 100, 0, 0
    Set s = doc.ActiveLayer.CreateRectangle(10, 100, 100, 0)
    s.Fill.UniformColor.CMYKAssign 0, 0, 100, 0
    Set s = doc.ActiveLayer.CreateRectangle(110, 100, 200, 0)
    s.Fill.UniformColor.CMYKAssign 0, 0, 0, 100

    With doc.PrintSettings
        ' 1. 选择 PostScript 输出设备
        .SelectPrinter "Device Independent PostScript File"
        .PrintToFile = True
        .FileMode = prnSeparatePlates    ' 每分色版一个文件
        .FileName = "C:\Output\Seps.ps"

        ' 2. PostScript 设置
        .PostScript.Level = prnPSLevel3
        .PostScript.AutoIncreaseFlatness = True
        .PostScript.AutoIncreaseFountainSteps = True
        .PostScript.ConformToDSC = True

        ' 3. 色彩分色设置
        With .Separations
            .Enabled = True
            .SpotToCMYK = True
            .AlwaysOverprintBlack = True
            .PreserveOverprints = True
            .EmptyPlates = False
        End With

        ' 4. 印前处理设置
        With .Prepress
            .CropMarks = True
            .ExteriorCropMarks = True
            .RegistrationMarks = True
            .RegistrationStyle = prnCorel
            .ColorCalibrationBar = True
            .PageNumbers = True
            .FileInfo = True
            .JobName = "客户稿件 - 印刷版"
            .InfoWithinPage = False
            .Mirror = False
            .Invert = False
        End With

        ' 5. 布局：居中，带出血
        With .Layout
            .Placement = prnPlaceCenterPage
            .UseBleedLimit = True
            .BleedLimit = 3          ' 3 mm 出血
        End With

        ' 6. 颜色选项
        With .Options
            .MarksToPage = False     ' 标记放在纸张边缘（非页面内）
            .UseColorProfile = True
            .FountainSteps = 256
        End With
    End With

    doc.PrintOut
    MsgBox "分色文件已输出到 C:\Output\"
End Sub
```

### 示例 2：桌面快速打印（多份、A4、彩色）

```vba
Sub QuickDesktopPrint()
    Dim prn As Printer
    Dim i As Integer

    ' 找一台可用的彩色打印机
    For i = 1 To Printers.Count
        If Printers(i).ColorEnabled And Printers(i).Ready Then
            Set prn = Printers(i)
            Exit For
        End If
    Next i

    If prn Is Nothing Then
        MsgBox "未找到可用的彩色打印机。"
        Exit Sub
    End If

    With ActiveDocument.PrintSettings
        Set .Printer = prn
        .SetPaperSize prnPaperSizeA4, prnPaperPortrait
        .Copies = 2
        .Collate = True
        .PrintRange = prnWholeDocument
        With .Options
            .ColorMode = prnModeFullColor
            .PrintBitmaps = True
            .PrintVectors = True
            .PrintText = True
            .UseColorProfile = True
        End With
        With .Layout
            .Placement = prnPlaceFitToPage
        End With
    End With
    ActiveDocument.PrintOut
    MsgBox "已发送到打印机：" & prn.Name
End Sub
```

### 示例 3：打印到 PDF（通过 PrintSettings 方式）

```vba
Sub PrintToPDF()
    With ActiveDocument.PrintSettings
        .SelectPrinter "Microsoft Print to PDF"
        .PrintToFile = True
        .FileMode = prnSingleFile
        .FileName = "C:\Output\" & _
                    Left(ActiveDocument.Name, InStrRev(ActiveDocument.Name, ".") - 1) & ".pdf"
        .Copies = 1
        .PrintRange = prnWholeDocument
        With .PostScript
            .Level = prnPSLevel3
            .pdfBookmarks = True
            .PDFHyperlinks = True
            .PDFStartup = prnPDFPageOnly
        End With
        With .Options
            .ColorMode = prnModeFullColor
            .RasterizePage = False
            .UseColorProfile = True
        End With
    End With
    ActiveDocument.PrintOut
    MsgBox "PDF 已输出到：" & ActiveDocument.PrintSettings.FileName
End Sub
```

### 示例 4：批量打印文件夹中的所有 CDR 文件

```vba
Sub BatchPrintFolder()
    Dim folder As String
    Dim fileName As String
    Dim doc As Document
    Dim count As Integer

    folder = "C:\MyDesigns\"
    fileName = Dir(folder & "*.cdr")
    count = 0

    Do While fileName <> ""
        Set doc = OpenDocument(folder & fileName)
        With doc.PrintSettings
            .SelectPrinter Printers.Default.Name
            .SetPaperSize prnPaperSizeA4, prnPaperPortrait
            .Copies = 1
            .PrintRange = prnWholeDocument
            .Options.ColorMode = prnModeFullColor
        End With
        doc.PrintOut
        doc.Close False         ' 不保存，直接关闭
        count = count + 1
        fileName = Dir()
    Loop
    MsgBox "共打印了 " & count & " 个文件。"
End Sub
```

| `PrintPostScript` | `ActiveDocument.PrintSettings.PostScript` |

| `PrintSeparations` | `ActiveDocument.PrintSettings.Separations` |
| `SeparationPlates` | `ActiveDocument.PrintSettings.Separations.Plates` |
| `SeparationPlate(i)` | `ActiveDocument.PrintSettings.Separations.Plates(i)` |
| `PrintTrapping` | `ActiveDocument.PrintSettings.Trapping` |
